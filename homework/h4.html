Answers to questions

  1. The box model is the term to describe the layout of HTML elements created using CSS. The box consists of the actual content and then the padding, border, and margin surrounding said content.

  2. A block level element is an HTML element that always starts on a newline and takes up the entire width of the line(s) it occupies.

  3. The h1 tag is by default rendered as a block-level element

  4. Inline elements are HTML elements that do not always start on newline, and it only takes up as much width of the screen as needed.

  5. The input tag is by default rendered as an inline element.

  6. We can include CSS in our pages in one of two ways.
    • Directly in our HTML by using a 'style' tag
    • Having all CSS code in it's own .css stylesheet and then linking it to our HTML file by using a 'link' tag.

  7. Selectors in CSS define the element(s) that will inherit the set of CSS rules within the function.

  8. Background color of the body is #0000FF
